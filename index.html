Overview

This project is a full stack android travel planning app for my WGU D424 Software Engineering Capstone Task 3.
This app is meant to allow users too:
- Create, edit, and delete vacations and excursions
- Search through records
- Generate reports
- Sync your data with the cloud
My Spring Boot backend was deployed to AWS Elastic Beanstalk. The backend connects to an AWS RDS MySQL database using the environment varialbles that were configured in Elastic Beanstalk.
My deployed app backend URL is:
http://capstone-app-env.eba-6epzk4je.us-east-2.elasticbeanstalk.com
The android app front end connects to the cloud backend through the ApiCleint.java address.
